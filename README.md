# OISP Student Ambassadors (OSA) - Landing Page

Welcome to the official repository for the OISP Student Ambassadors landing page. This project serves as the digital front door for OSA, helping to promote a positive image of OISP students and acting as a bridge between the student body, the administration, and potential employers.

## 🚀 Features

* **Modern Hero Section:** Engaging introduction with a clear call-to-action.
* **Sticky Navigation:** A persistent top navigation bar for seamless scrolling and easy access to different sections.
* **Team Showcase:** Clean profile cards highlighting the club's leadership.
* **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/)
* **Library:** [React](https://reactjs.org/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repo:
   ```bash
   git clone [https://github.com/your-username/osa-landing-page.git](https://github.com/your-username/osa-landing-page.git)
   ```
2. Navigate into the project directory:
   ```bash
   cd osa-landing-page
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 📁 Project Structure

* `/components` - Reusable UI components (Navbar, Hero, TeamCard, Footer).
* `/app` - The main application routes and pages.
* `/public` - Static assets like images and icons.
